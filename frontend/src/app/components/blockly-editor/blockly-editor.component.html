Block Code

The "Block Code" section is used to drag and drop customizable code blocks that will determine what the robot simulator does after the start button is pressed. Drag and drop a block to the trash can to delete it from the workspace.

XML

The "XML" section displays the XML code that your current blocks produce. To save your current block configuration, copy the XML and save it. Paste or type XML code into this section and click "Update Code Blocks" to restore the blocks.
